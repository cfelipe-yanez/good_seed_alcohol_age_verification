# good_seed_alcohol_age_verification
Este es el repositorio en el que se creará todo para el proyecto del sprint 18 del curso de Triple Ten de Visión Artificial
# Good Seed - Alcohol Age Verification

## Descripción del proyecto
Este proyecto tiene como objetivo evaluar si la ciencia de datos y los métodos de visión artificial pueden ayudar a la cadena de supermercados **Good Seed** a cumplir con las leyes sobre la venta de alcohol, evitando vender alcohol a personas menores de edad.

El proyecto consiste en construir y evaluar un modelo de visión artificial capaz de estimar la edad de una persona a partir de fotografías en el área de pago de las tiendas.

## Objetivos
- Realizar un análisis exploratorio de los datos.
- Preprocesar las imágenes para entrenamiento y prueba del modelo.
- Entrenar y evaluar un modelo de red neuronal para estimar la edad.
- Combinar código, resultados y conclusiones en un Jupyter Notebook.
- Entregar un informe con hallazgos y conclusiones sobre la evaluación del modelo.

## Datos
El conjunto de datos consiste en fotografías de personas con su respectiva edad. Los datos serán usados para:
- Entrenamiento del modelo de red neuronal.
- Evaluación del desempeño del modelo.
